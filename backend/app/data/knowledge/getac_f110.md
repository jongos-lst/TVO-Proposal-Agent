# Getac F110 — Fully Rugged Tablet

## Overview
The Getac F110 is an 11.6-inch fully rugged Windows tablet built for mobile workers who need a portable yet powerful field device. Ideal for utilities, transportation, logistics, and field inspection.

## Key Differentiators
- **Hot-Swappable Battery**: Change battery without powering down, critical for continuous field operations.
- **Sunlight-Readable 1000 nit Display**: Touchscreen works with gloves and in wet conditions.
- **Vehicle Dock Compatible**: Seamlessly transitions between field and vehicle use.
- **Digitizer Pen Support**: Precise input for forms, annotations, and signatures.

## Performance Specifications
- Intel Core i7-1265U, up to 32GB RAM, up to 1TB SSD
- MIL-STD-810H, IP66 rated
- Dedicated GPS module for accurate positioning
- 4G LTE, Wi-Fi 6E connectivity
- Optional barcode reader and RFID

## Competitive Edge vs Consumer Tablets
- Purpose-built for field use — not a consumer tablet in a case
- 2.5% annual failure rate vs 12-15% for consumer tablets in field conditions
- Direct sunlight visibility eliminates the #1 complaint of field workers using iPads
- Vehicle dock integration provides true mobile-to-vehicle computing continuity
