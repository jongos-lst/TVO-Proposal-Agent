# Getac X600 — Fully Rugged Mobile Server Laptop

## Overview
The Getac X600 is a 15.6-inch fully rugged performance laptop designed for high-compute workloads in extreme environments. With up to 128GB RAM, Intel Xeon processor option, and optional NVIDIA RTX A4500 GPU, it serves as a mobile workstation and field server.

## Key Differentiators
- **Mobile Server Performance**: Intel Xeon processor + up to 128GB RAM + NVIDIA RTX A4500 (16GB) — runs CAD, simulation, and AI workloads in the field.
- **Massive Storage**: Up to 3 user-removable SSDs totaling 6TB — enough for large datasets, mapping, and video.
- **Dual 2.5GBASE-T Ethernet**: Direct high-speed network connectivity for server roles.
- **Full Legacy Port Array**: Serial, VGA, DisplayPort, and HDMI — connects to older industrial and defense equipment without adapters.
- **IP66 + MIL-STD-810H + MIL-STD-461G**: Complete military and environmental certifications.

## Performance Specifications
- Intel Core i7-11850HE / Xeon W-11865MRE vPro
- Up to 128GB DDR4 RAM
- Optional NVIDIA Quadro RTX3000, GTX1650, or RTX A4500
- Dual hot-swappable 6900mAh batteries
- Wi-Fi 6E, Bluetooth 5.2, optional 4G LTE

## TCO Advantages
- 2% annual failure rate vs 15%+ for commercial workstations in field
- Replaces both laptop + external server in mobile command scenarios
- Bumper-to-bumper warranty on a $5,499 device eliminates repair cost risk
- 7+ year lifespan eliminates multiple replacement cycles
