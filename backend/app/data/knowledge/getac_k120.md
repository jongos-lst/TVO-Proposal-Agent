# Getac K120 — Fully Rugged Convertible Tablet

## Overview
The Getac K120 is a 12.5-inch fully rugged tablet with 7 usage modes (laptop, convertible, presentation, tablet, handle, tilt, docking). Designed for field service workers who need maximum flexibility.

## Key Differentiators
- **7 Usage Modes**: Most versatile rugged device — adapts to any field scenario without additional hardware.
- **1200 nit LumiBond 2.0**: High-brightness sunlight-readable display with Getac's latest display bonding.
- **Dual Hot-Swappable Batteries**: Zero-downtime battery changes in the field.
- **5 Programmable Buttons**: Custom-mapped to field applications for one-touch workflow access.
- **IP66 + MIL-STD-810H**: 6-foot drop resistant, fully sealed against dust and water.

## Performance Specifications
- Intel Core i7-1365U vPro (13th Gen), up to 64GB DDR5
- Up to 2TB PCIe NVMe SSD
- Wi-Fi 6E, Bluetooth 5.3, optional 5G Sub-6
- Dual Thunderbolt 4, Gigabit Ethernet
- 8MP rear + 5MP front cameras

- 2% annual failure rate
- Replaces separate laptop and tablet purchases
- Bumper-to-bumper warranty covers accidental damage
- Detachable keyboard dock extends functional lifespan
